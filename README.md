------
클론 후 사용법
------

### 1. 파이썬 가상환경을 설치한다.
      $ py -3.9 -m venv qtenv  
### 2. 가상환경을 활성화 한다.
      $ qtenv\Scripts\activate
### 3. 파이썬 모듈 가상환경에 설치(PyQt5, pymysql)
      $ pip install PyQt5
      $ pip install PyQt5-tools
      $ pip install pymysql
### 4. 프로그램 실행




------
사용자 요구사항
------

### Python 3.9.xx 설치, MySQL 서버 설치, 데이터베이스 스키마(clothes) 생성, 테이블(users, management) 생성
